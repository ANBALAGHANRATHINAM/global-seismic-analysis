#  Global Seismic Analysis

## Project Description
This is a mini learning project focused on analyzing earthquake data.
The goal is to practice Python and SQL using real seismic datasets.

## Objectives
- Understand earthquake data
- Practice Python for data cleaning and analysis
- Practice SQL queries on earthquake data
- Learn basic data visualization

## Tools Used
- Python
- SQL
- Pandas
- Jupyter Notebook
- GitHub
-streamlit
